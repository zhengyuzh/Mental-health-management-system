# 系统使用说明
该系统是在若以框架的基础上进行的开发，前端使用的是Thymeleaf + BootStrap进行开发
## 环境配置说明
- JDK 1.8
- Mysql 5.7 
- Maven 3.3.9
- IDEA 2020
